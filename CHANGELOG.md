# Changelog

All notable changes to the Mishu Theme extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-08

### Added

- Initial release of Mishu Theme
- **Mishu Dark**: Balanced default theme with warm, low-contrast colors
- **Mishu Dark Soft**: Ultra-low contrast variant for sensitive eyes
- **Mishu Midnight**: Deep black variant optimized for OLED displays

### Features

- Semantic syntax highlighting based on code importance
- Complete UI theming (editor, sidebar, terminal, panels, etc.)
- Support for 15+ programming languages
- Bracket pair colorization
- Git decoration colors
- Debug and testing colors
- Comprehensive tokenization for:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS/SCSS
  - JSON/YAML
  - Markdown
  - Go
  - Rust
  - C/C++
  - SQL
  - Shell scripts
  - And more...

### Design Principles

- Eye comfort prioritized over vibrancy
- Importance-based color assignment
- Reduced visual noise (dimmed punctuation)
- Warm color temperature to reduce eye strain
- Consistent visual hierarchy across all UI elements
